# Minimalist Calculator Classic

This project is a basic calculator built using **HTML, CSS, and JavaScript**. It provides a user-friendly interface for performing standard arithmetic operations.

## Preview
![Minimalist Calculator Classic Preview](https://github.com/mihaiapostol14/Minimalist-Calculator-Classic/blob/7ba114cf9ee94c8a1a1f4846cf70f94a2782afc5/assets/preview.png)


## Features
- **Basic Operations:** Addition, Subtraction, Multiplication, Division, Percentage,Reciprocal,Square Root.
- **Clear and Delete Functions:** `C` clears all input, `CE`  `⌫` deletes the last character.
- **Real-time Display Update:** Updates the display as users click buttons.

## Technologies Used
- **HTML**: For structuring the calculator layout.
- **CSS**: For styling and responsiveness.
- **JavaScript**: For handling button interactions and calculations.

## Getting Started

To get a local copy of this project up and running, use the following command:

```bash
git clone https://github.com/mihaiapostol14/Minimalist-Calculator-Classic.git
cd Minimalist-Calculator-Classic
```

## Usage
1. Click on the buttons to input numbers and operations.
2. Press `=` to evaluate the expression.
3. Use `C` to clear all input or `CE`  `⌫` to delete the last character.

## How It Works
The JavaScript script listens for button clicks and updates the calculator's input field accordingly. It uses `Dynamic Function Construction` (with safety replacements) to perform calculations.
